-------------------
Date exposed filter
-------------------

This Drupal 7 module allows you to add an exposed date filter so you can filter
nodes given their changed date. User will be able to specify a "from" and "until"
date. This module is compatible with normal and search api views.

IMPORTANT NOTE: This module is meant to be used for programmers as a base to
start with. However it may be a plug & play in your case, just test it.
